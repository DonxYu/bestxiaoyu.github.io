注册

Create Droplet

注册完成之后进入Digitalocean首页 右上角Create Droplet

选择Ubuntu系统 14.04系统

选择自己需要的套餐 个人用的话$5就够了

机房地址选择，San Francisco比较稳定，速度相对也较快，推荐

hostname自己随意填吧

存储公钥，没有公钥的话每次登陆需要输入密码，密码DO的邮件里会体现
公钥生成方式: 打开终端或者putty

vim ~/.ssh/id_rsa.pub 复制里面内容放进SSH Key content保存

到此配置已完成，点Create吧。

接下来终端或者putty登陆VPS

复制一下命令安装shadowsocks
